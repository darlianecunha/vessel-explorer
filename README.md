# Vessel Explorer

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20708997.svg)](https://doi.org/10.5281/zenodo.20708997)

Interactive web explorer for a fleet of **1,570 IMO-validated vessels with complete records**, enriched and classified from the **EU MRV 2024** emissions dataset (Regulation EU 2015/757). Only vessels with ship type, DWT, efficiency grade and flag all present are included.

Live data layer: `data.json` (generated by `../build_dataset.py`).

## Features

- Search by vessel name or IMO number
- Filter by **ship type**, **DWT size class**, **emissions efficiency grade (A–E)** and **flag / port of registry**
- Sortable table, summary cards and charts
- Each vessel links to its MarineTraffic page (by IMO)

## Classifications

| Axis | Source | Notes |
|---|---|---|
| Ship type | EU MRV | Bulk carrier, oil/chemical tanker, container, etc. |
| DWT size class | Validated DWT | Small / Handysize / Handymax–Supramax / Panamax / Large / Very Large |
| Efficiency grade A–E | EU MRV CO₂ intensity | Relative CO₂-per-distance band, ranked **within ship type**. Not the official IMO CII. |
| Flag / registry | EU MRV | Port of registry, normalised |

All 1,570 vessels carry full MRV attributes (type, port, emissions, DWT and efficiency grade); incomplete records were removed.

## Data dictionary (`data.json`)

| Field | Description |
|---|---|
| `name` | Vessel name |
| `imo` | IMO number (unique identifier) |
| `ship_type` | EU MRV ship type |
| `dwt` | Deadweight tonnage (validated) |
| `dwt_class` | Size class derived from DWT |
| `efficiency_grade` | A–E band, derived from CO₂-per-distance within ship type |
| `flag` | Port of registry (normalised) |

## Deploy to Vercel

This is a static site — no build step.

```bash
npm i -g vercel
vercel        # preview
vercel --prod # production
```

Or import the GitHub repo at vercel.com → New Project → Framework preset: **Other**.

## Files

- `index.html` — single-file explorer (Chart.js from CDN)
- `data.json` — dataset (1,570 vessels)
- `vercel.json` — static config (clean URLs, caching)
